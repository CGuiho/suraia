# PillsInput

An input container styled to contain multiple pills alongside an input field. It acts as a wrapper styling component that matches standard input sizes and states (focused, error, disabled).

## Dependencies
- Input

## AI Translation Notes
- Implements ARIA `role="group"` on the wrapping container.
- Focus is visually reflected on the container when the nested input is focused.
- Supports identical visual sizes (`xs`, `sm`, `md`, `lg`, `xl`) and validation/disabled states as the standard text input component.
