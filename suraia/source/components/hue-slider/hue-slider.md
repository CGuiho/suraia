# HueSlider

A slider component designed to select a hue value (0-360) from the color spectrum.

## AI Translation Notes
- Track is styled with a multi-stop color gradient of the entire HSV/HSL hue spectrum (from red through yellow, green, cyan, blue, magenta, back to red).
- The slider value goes from 0 to 360, representing the hue angle in degrees.
- Drag events calculate the percentage along the track to compute the selected hue value.
- Keyboard support matches a standard slider but ranges from 0 to 360.
