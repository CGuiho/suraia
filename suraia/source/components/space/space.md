# Space

An empty spacing element that adds vertical or horizontal whitespace between elements.

## Dependencies
None — Tier 1 Primitive.

## Usage
Space accepts `h` (height) and `w` (width) attributes which map to inline styles. Use spacing token values.

## AI Translation Notes
- Renders as an empty `<div>` with the specified dimensions.
- `min-height`/`min-width` are set equal to `height`/`width` to prevent collapse.
