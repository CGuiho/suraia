# AlphaSlider

A slider component designed to modify the alpha (transparency) channel of a color.

## AI Translation Notes
- Renders a checkered background overlaid with a gradient from transparent to the opaque target color.
- CSS uses a double-layered layout: checkers background for transparency representation, and color gradient overlay.
- Drag actions compute percentage along the track to determine alpha values (0 to 1).
- Accessible keyboard navigation is supported using standard arrow keys.
