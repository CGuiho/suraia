# List

List provides semantic ordered and unordered list anatomy with spacing, padding, and icon alignment guidance.

## Dependencies

None - Tier 1 Primitive.

## AI Translation Notes

- Generate `ul` for unordered lists and `ol` for ordered lists.
- Keep native list semantics unless custom icons require `list-style: none`.
- Use item icons as decorative by default unless they communicate additional meaning.
- Map spacing to `data-suraia-spacing` and CSS variables instead of hard-coded margins.
