# Indicator

Renders a small status badge or dot on the corner of another element.

## AI Translation Notes
- Wraps the target child inside a `.suraia-indicator-root` container set to `position: relative`.
- Renders the indicator dot absolutely positioned relative to the root.
- The `processing` attribute activates a pulsing visual ripple using an `@keyframes` pulse animation.
- Offset adjustments or label text can be rendered inside the indicator dot.
