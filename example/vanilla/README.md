# Suraia Vanilla TypeScript Example

This is a generated-target example for Vanilla HTML, CSS, and TypeScript. It demonstrates how an AI assistant can read Suraia blueprints and produce local application code without importing Suraia at production runtime.

## Run

```bash
bun install
bun run dev
```

The dev server defaults to `http://localhost:4101`.

## Verify

```bash
bun run build
```

## What This Demonstrates

- Multi-page Vanilla TypeScript UI rendered from local state.
- Generated-style components: buttons, dialog, tabs, form fields, progress, status states, and data rows.
- Loading, empty, error, success, toast, and dialog interaction states.
- Suraia-style classes and `data-suraia-*` attributes owned by the example app.
