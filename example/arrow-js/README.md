# Suraia Arrow.js Example

Status: planned.

This directory is reserved for a future Arrow.js target. The intended output is an Arrow app generated from the same Suraia blueprint contracts used by the Vanilla and React examples.

## Planned Direction

- Translate `.structure.html` files into Arrow templates.
- Translate controller behavior into Arrow reactive state and events.
- Reuse Suraia CSS class and `data-suraia-*` conventions.
- Keep final generated files owned by the target project, with no production Suraia runtime import.
